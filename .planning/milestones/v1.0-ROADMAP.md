## Phases

### Phase 1: Foundation — Project Setup, Ethics & Browser
**Status:** Complete
**Goal:** Bootstrap the Bun+TypeScript project with all utilities, ethics infrastructure, and browser connection established so subsequent phases can scrape immediately.

- TypeScript strict mode configured (tsconfig.json)
- Job interface with all fields (src/types.ts)
- Utilities: delay, retry/backoff, robots.txt checker, relative date parser, company normalizer, UA rotation pool
- Browser module: Camoufox port detection → Playwright connect → local launch → fetch-only fallback chain
- Ethics: 3s minimum delay, max 3 retries, max 10 pages per site, robots.txt checking

### Phase 2: Tier 1 Scraping + Data Processing Pipeline
**Status:** Complete
**Goal:** Scrape all 4 Tier 1 sources (Internshala, Wellfound, RemoteRocketship, Unstop) and build the complete data processing pipeline.

- Internshala: fetch + cheerio, pagination up to 10 pages, 100+ jobs per run
- Wellfound: Camoufox anti-detect browser with fetch fallback; CF-challenge detection
- RemoteRocketship: fetch + cheerio, targets entry-level + rust/typescript filters
- Unstop: Angular HTML scraper + REST API fallback
- Filter pipeline: location + experience + skills scoring
- Dedup via SHA-256 hostname-based key
- HIGH MATCH detection via word-boundary regex (Rust/Solana/TypeScript/Next.js)
- Output: job_opportunities.md with progress reporting every 25 jobs

### Phase 3: Tier 2 Scraping (LinkedIn, Cutshort, Himalayas)
**Status:** Complete
**Goal:** Add all 3 Tier 2 sources with graceful handling for LinkedIn's anti-bot measures.

- LinkedIn: guest jobs API (no auth required), graceful skip on 403/block
- Cutshort: fetch + cheerio, 6 search URLs across tech stacks, 5s delay for rate-limit
- Himalayas: fetchRendered (Playwright → plain fetch fallback) for React SPA, 6 URL variants

### Phase 4: Tier 3 Web3/Rust/Solana Scraping
**Status:** Complete
**Goal:** Add both Web3-focused sources and ensure Rust/Solana jobs are discovered and properly HIGH MATCH flagged.

- SolanaJobs: 19 jobs returned, all flagged HIGH MATCH (Solana keyword match)
- Web3.career: Web3 career scraper
- CryptocurrencyJobs: crypto/Web3 jobs
- YCombinator: YC Work at a Startup
- All 4 registered in SCRAPERS map
- Bun build clean: 260 modules, 0 errors

### Phase 5: Output Formatting, Sorting & Final Run
**Status:** Complete
**Goal:** Generate the final job_opportunities.md with proper sorting, application tracker table.

- formatter.ts generates markdown checklist with job entry template
- Sort: HIGH MATCH first, then by recency descending
- Full pipeline: filter → dedup → sort → format → write
- Application tracker table appended
- 254 total jobs, 22 HIGH MATCH
- Exit code 0 on full run

| Markdown checklist output | - [ ] format enables tracking in any markdown viewer |
| 🔥 HIGH MATCH flag | Rust/Solana/TypeScript/Next.js keyword scan |
| SHA-256 dedup | Hash of company + title + URL prevents duplicates |
